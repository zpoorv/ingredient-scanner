# Security Policy

## Supported Versions

Only the latest `main` branch and the most recent store build should be considered supported for security fixes.

## Reporting a Vulnerability

Please do not open public GitHub issues for security problems.

Report security concerns to:

- `support@inqoura.app`

Include:

- a short summary of the issue
- steps to reproduce it
- affected screen, endpoint, or feature
- screenshots or logs if helpful

We will review reports as quickly as possible and coordinate a fix before public disclosure when appropriate.

## Scope Notes

Inqoura relies on Firebase, Google Play Billing / RevenueCat, and AdMob. If a report is clearly caused by one of those platforms, please still send it so we can confirm impact and route it correctly.
